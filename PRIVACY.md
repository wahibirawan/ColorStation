# Privacy Policy - ColorStation

## Data Collection

ColorStation does **not** collect personal data.

## Data Stored Locally

ColorStation stores only in `chrome.storage.local` on the user's device:

- Settings:
  - default color format
  - default sampling size
  - default grid visibility
  - native picker preference
  - loupe zoom preference
- Color history (up to 5 items):
  - HEX, RGB, HSL values
  - timestamp
  - pin/favorite state

## Network Usage

ColorStation makes no external network requests for telemetry or analytics.

- no analytics
- no telemetry
- no trackers
- no third-party API calls

## Screen Capture Handling

When picker mode is explicitly started by the user, ColorStation captures only the visible tab area to sample pixels. Capture data is processed locally in memory and is not transmitted.

## Third Parties

No third-party services are used.
